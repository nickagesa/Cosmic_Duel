# Cosmic Duel 🚀🔥

**Cosmic Duel** is a fast-paced 2D space battle game where two players engage in an intergalactic shootout. The game features smooth animations, immersive sound effects, and an exciting competitive mode.

## Features
- 🚀 **Two-player mode**: Battle against a friend!
- 🎶 **Background music & sound effects**: Experience an immersive atmosphere.
- 💥 **Dynamic bullet collisions**: Test your reflexes and aiming skills.
- 🌌 **Scrolling space background**: Feel the depth of space combat.
